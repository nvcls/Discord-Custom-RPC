Custom discord RPC status using python UI. Requires discord desktop app to be open in order to function properly.

> ‚ùï The RPC will take the header of your clients name (discord bot), choose the name you want to appear.
### Client ID
You must chance the client ID in order for it to work, to do this complete the steps below:
1. Create a discord bot using the [discord developer portal](https://discord.com/developers/applications)
2. Once you have setup the bot, copy the bot's client ID (you don't need to do anything to the bot)
3. Find the client id in the link of the tab open <img width="772" height="50" alt="image" src="https://github.com/user-attachments/assets/b4dfe790-1c03-4223-ace8-c1d52813ed0c" />

4. Copy it into this line of the code
CLIENT_ID = "client id"

### Installing the dependencies
`pip install pypresence PyQt5 time sys traceback`

### Customising
Use the UI that opens to customise how the RPC looks, to add images, you must do this inside of the "Art Assets" tab of the developer portal.
- Add Image(s) under the Rich Presence Assets header
- The name of the image is what you should enter into the input box
- All other info can be typed in
